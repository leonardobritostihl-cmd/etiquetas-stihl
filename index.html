Visão Geral

Esta ferramenta gera etiquetas de envio no formato 11 × 9 cm em PDF, prontas para impressão. Preencha os dados nas abas, configure o nome do arquivo e clique em Gerar Etiqueta PDF.

Passo a Passo
1
Aba — Destinatário

Preencha os dados de quem receberá o pacote: nome, endereço completo, bairro, município, UF, CEP, CPF/CNPJ e telefone. Todos os campos são obrigatórios.

O CEP informado é usado para gerar o código de barras na etiqueta — digite no formato 00000-000.
2
Aba — Envio

Informe os dados do despacho: número da nota fiscal, nome da transportadora, peso total em quilogramas e a contagem de volumes (ex.: 1 / 3 indica o primeiro de três volumes).

3
Aba — Arquivo

Defina o nome do arquivo PDF que será baixado. Não é necessário adicionar a extensão .pdf — ela é inserida automaticamente. Se o campo ficar em branco, o arquivo será salvo como etiqueta.pdf.

4
Gerar Etiqueta PDF

Clique no botão laranja Gerar Etiqueta PDF na barra inferior. O sistema valida todos os campos — se algum estiver vazio, uma lista de pendências será exibida. Após confirmar, o PDF é gerado e baixado automaticamente.

5
Limpar Campos

Use o botão Limpar campos para apagar todos os dados e iniciar um novo preenchimento. Uma confirmação será solicitada antes de apagar.

Dicas de Impressão
  • Abra o PDF gerado e imprima em papel de etiqueta adesivo ou folha comum cortada no tamanho 11 × 9 cm.
  • Nas configurações de impressão, desative o redimensionamento automático (Tamanho real / 100%) para garantir as dimensões corretas.
  • Recomenda-se verificar o código de barras com um leitor antes de despachar o pedido.
